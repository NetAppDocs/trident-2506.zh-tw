= 部署範例應用程式
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
建立儲存等級和 PVC 後，即可將光電模組安裝到艙體上。本節列出了將 PV 連接到 pod 的範例命令和設定。

.步驟
. 將音量旋鈕安裝在一個音控器中。
+
[source, console]
----
kubectl create -f pv-pod.yaml
----
+
以下範例展示了將PVC連接到艙體的基本配置：*基本配置*：

+
[source, yaml]
----
kind: Pod
apiVersion: v1
metadata:
  name: pv-pod
spec:
  volumes:
    - name: pv-storage
      persistentVolumeClaim:
       claimName: basic
  containers:
    - name: pv-container
      image: nginx
      ports:
        - containerPort: 80
          name: "http-server"
      volumeMounts:
        - mountPath: "/my/mount/path"
          name: pv-storage
----
+

NOTE: 您可以使用以下方式監控進度 `kubectl get pod --watch`。

. 確認卷已掛載到 `/my/mount/path`。
+
[source, console]
----
kubectl exec -it pv-pod -- df -h /my/mount/path
----


[listing]
----
Filesystem                                                        Size  Used  Avail Use%  Mounted on
192.168.188.78:/trident_pvc_ae45ed05_3ace_4e7c_9080_d2a83ae03d06  1.1G  320K  1.0G   1%   /my/mount/path
----
現在您可以刪除該 Pod 了。  Pod 應用將不復存在，但卷將保留。

[source, console]
----
kubectl delete pod pv-pod
----
